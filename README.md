# RealLight

A tool that ingests an image sequence of a real scene, analyses it, and produces a starting-point light rig inside Katana — physical lights placed in 3D with colour, direction, type, and a rough intensity. The output is a strong starting point for the artist to refine, not a finished result.

See [docs/design.md](docs/design.md) for the full architecture, tool choices, and open questions.
